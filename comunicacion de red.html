Capas de modelo OSI

Descripcion
Capa Fisica La capa física es donde el flujo de bits sin procesar se transmite físicamente a través de un medio físico.
Capa de enlace de datos La capa de enlace de datos rompe los datos que se van a transmitir en tramas para su transmisión en la capa física. También administra conexiones entre dos nodos diferentes
Capa de red En la capa de red, el enfoque se expande desde un enlace punto a punto para incluir muchos nodos interconectados dentro de una red.
Capa de transporte La capa de transporte es la primera de cuatro capas “host” y el resto se conoce como capas “multimedia”.
Capa de sesión La capa de sesión administra sesiones entre nodos y actúa sobre la PDU de “datos”. La administración de sesiones incluye configuración, autenticación, terminación y reconexiones.
Capa de presentación La capa de presentación es la principal responsable de traducir datos de la red a los formatos esperados por una aplicación. Por ejemplo, la codificación y el cifrado de datos se gestionan en la Capa 6.
Capa de aplicación La capa de aplicación incluye protocolos diseñados para usuarios finales. Por ejemplo, HTTP es un protocolo de capa 7 diseñado para transmitir datos entre un servidor web y un cliente.
